# dmdw-project

### Client Setup

1. Install dependencies
   
```bash
  $ npm i
```

2. Run the client

```bash
  $ npm start
```

### Server Setup

1. Install dependencies

```bash
  $ pip install -r requirements.txt
```
   
2. Run the server

```bash
  $ uvicorn --app-dir="./app" --reload main:app
```
